# Typescript Guidelines

Let's keep all the JS assets to this path.

## Naming classes

Use `PascalCase` for classes.

```ts
class YswProduct {
    constructor() {}
}

class YswGlobal {
    constructor() {}
}
```

## Naming functions, variables, methods, etc

Use `camelCase` for functions, variables methods and others.

```ts
function getProduct() {}

const productPrice = () => {}
```
